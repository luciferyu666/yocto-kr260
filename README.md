# yocto-kr260

### 簡介 ###

Yocto-KR260 是基於 Yocto Project 的自訂建置系統，專為 KR260 平台量身打造。本儲存庫包含了必要的層（Layers）、配方（Recipes）及配置檔，主要目的在建置一個優化的 Linux 映像檔，適用於 KR260。

KR260 是一款專為 AI 和工業應用設計，結合了高性能的 FPGA 和靈活的 ARM 處理器的客製化 KR260  Kria 開發板 BSP (Board Support Package) 韌體專案。
透過 Yocto Project，Yocto-KR260 提供了一個靈活且強大的環境，便於開發具精細控制的嵌入式 Linux 系統。

### 功能 ###
- 自訂 Linux 映像：專為 KR260 平台設計。
- 優化的內核配置：提升效能與穩定性。
- 預配置的軟體套件：包含必要的工具與服務。
- 支援特定功能：例如網路、GUI 等。
- 易於自訂：透過 Yocto 層（Layers）擴展與修改。
